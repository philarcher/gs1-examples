GS1 Web Vocabulary Examples

This began as a clone of the examples at https://data.gs1.org/example/.

Introduction

This is a jumping off point for example pages that include structured data that use the GS1 Web vocabulary. In a real context, each page would include human-readable text showing the information that in these pages is only provided in the JSON-LD embedded in the page.

Key features of the data:

  • schema.org terms used wherever possible
  • GS1 terms used to provide a richer description

The examples:

  • Breakfast cereal
  • Cheese
  • Milk*
  • Mixed nuts*
  • Picked onions
  • Tangerines*

* denotes pages that represent a retailer's web page so these are offers. Other pages represent a brand owner's product description page.

On these example pages, the image is linked to the relevant page on the retailer/brand owner's live website.
